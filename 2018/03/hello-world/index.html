Hello World!

三月 14, 2018 发布在 Geek

Leon117的个人博客终于上线啦！

第一篇blog内容当然是：Hello World!

建个人博客初衷是把若干年来收集的各种有趣好玩的东西记录归档。以前虽然在移动硬盘做过整理，但硬盘现在基本常年吃灰，也不知道什么时候就可能物理损坏了。Chrome上的书签地址虽然存得密密麻麻，想看的时候网页却不在了，于是索性自己整理，做一个静态网站，托管到Github上，这样方便了自己，也能分享给别人，这个项目因此诞生。

那么先简单介绍一下本站：这是一个用Hugo生成的静态网站。Hugo是用 go 语言实现的轻量级静态网站生成器，开源，简单易用，快速部署。 Linux 和 Mac 可以用控制台来运行， Win10 建议配合 PowerShell 来使用，在本地建站调试好后，用 git push到Github页面上，博客站就建好了。以后只要用 Markdown 格式的文件记博客，然后push，就OK了。未来有空的时候，我也会专门写一篇Hugo的文章，供有兴趣自己建站的朋友参考。
